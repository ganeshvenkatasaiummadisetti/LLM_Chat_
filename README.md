# LLM_Chat_
A simple LLM chat application built with Streamlit that connects to the Groq API to generate responses using the LLaMA 3.1 8B Instant model.

This project demonstrates how to build a lightweight chat interface with session-based conversation memory.

🚀 Features

Simple chat UI using Streamlit

Uses Groq LLM API

Maintains conversation history with st.session_state

Instant responses using LLaMA-3.1-8B-Instant

Environment variable support using .env

🛠️ Tech Stack

Python

Streamlit

Groq API

Python Dotenv
